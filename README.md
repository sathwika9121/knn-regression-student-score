# KNN Regression Student Score Prediction

## 📌 Project Overview

This project demonstrates the implementation of the K-Nearest Neighbors (KNN) Regression algorithm using Python, Scikit-learn, and Streamlit to predict student math scores.

## 📂 Dataset

The project uses the Students Performance Dataset.

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## 🚀 Features

* Data Preprocessing
* Label Encoding
* Feature Scaling
* KNN Regression Model
* Student Score Prediction
* Interactive Streamlit UI

## 📊 Algorithm Used

K-Nearest Neighbors Regression (KNN Regression)

KNN Regression predicts continuous numerical values based on nearest neighboring data points.

## 📈 Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

## ▶️ How to Run

Install required packages:

```bash
pip install pandas numpy scikit-learn streamlit
```

Run the Streamlit app:

```bash
python -m streamlit run app.py
```

## 📌 Project Workflow

1. Import Dataset
2. Data Preprocessing
3. Train-Test Split
4. Feature Scaling
5. Build KNN Regression Model
6. Train Model
7. Predict Student Math Score
8. Evaluate Model Performance
